# Yabai Focus Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Add `Focus App` command for searching open `yabai` windows from Raycast.
- Show app icons and separate rows for multiple windows from the same app.
- Exclude Raycast's own command palette from the window list.
- Focus selected windows through `yabai`, with an `open -a` fallback when direct focus fails.
- Add refresh and copy-command actions for quick recovery and debugging.
